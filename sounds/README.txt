AUDIO FILES FOR POMODORO TIMER
===============================

This directory is for optional audio notification files.

Suggested files:
- bell.mp3 (for work session end)
- break.mp3 (for break session end)

You can:
1. Record your own notification sounds
2. Download free sounds from:
   - Freesound.org
   - Zapsplat.com
   - Adobe Stock Audio (free tier)
   - YouTube Audio Library

Note: The timer works without audio files! 
It will generate beep sounds using Web Audio API and show browser notifications as fallback.

Recommended audio length: 1-3 seconds
Recommended format: MP3 or WAV
